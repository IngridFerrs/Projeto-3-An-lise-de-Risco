# Projeto-3-An-lise-de-Risco
O objetivo da análise é identificar o perfil de clientes com risco de inadimplência, montar uma pontuação de crédito a través da análise de dados e avaliar o risco relativo, possibilitando assim, classificar os clientes e futuros clientes em diferentes categorias de risco com base na sua probabilidade de inadimplência
